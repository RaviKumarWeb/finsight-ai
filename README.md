# FinSight AI 🔍

A production-grade multi-agent financial research platform built with LangGraph and FastAPI. Input any company name and receive a comprehensive investment research report generated by 5 specialized AI agents working in coordination.

## Architecture
```
User Request
     ↓
FastAPI (Async + ThreadPoolExecutor)
     ↓
LangGraph Orchestrator
     ↓
┌─────────────────────────────────────┐
│  News Agent      → Tavily Search    │
│  Financial Agent → yfinance API     │
│  Risk Agent      → LLM Analysis     │
│  Report Agent    → Draft + Retry    │
│  Critic Agent    → Quality Check    │
└─────────────────────────────────────┘
     ↓
SQLite Database (Full Audit Trail)
     ↓
Verified Research Report
```

---

## Key Features

- **5 Specialized Agents** — each with single focused responsibility
- **Optimized Pipeline** — only 5 LLM calls per research (down from 8+)
- **Smart Retry Logic** — Critic sends specific errors back to Report Agent for targeted fixes
- **Real Financial Data** — yfinance direct API (zero LLM cost for ticker resolution)
- **Confidence Scoring** — every report scored 0.0-1.0 before delivery
- **Full Audit Trail** — every agent action logged including failures
- **Async FastAPI** — ThreadPoolExecutor prevents event loop blocking
- **Docker Ready** — containerized for consistent deployment anywhere
- **Railway Deployed** — live on the internet

---

## How The Retry Loop Works
```
Report Agent writes draft
      ↓
Critic Agent checks every number against source data
      ↓
Confidence ≥ 0.8? → Deliver to user ✅
Confidence < 0.8? → Send specific errors to Report Agent
                  → Report Agent rewrites fixing those errors
                  → Critic checks again (max 2 retries)
```

## Quick Start — Local
```bash
# Clone
git clone https://github.com/RaviKumarWeb/finsight-ai.git
cd finsight-ai

# Install
pip install -r requirements.txt

# Configure
cp .env.example .env
# Add your API keys to .env

# Run
uvicorn api.main:app --reload
```

## Quick Start — Docker
```bash
git clone https://github.com/RaviKumarWeb/finsight-ai.git
cd finsight-ai

cp .env.example .env
# Add your API keys

docker build -f docker/Dockerfile -t finsight-ai .
docker run -p 8000:8000 --env-file .env finsight-ai
```
