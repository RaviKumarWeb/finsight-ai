# FinSight AI 🔍

A production-grade multi-agent financial research platform built with LangGraph and FastAPI. Input any company name and receive a comprehensive investment research report generated by 5 specialized AI agents working in coordination.

## Architecture
```
User Request
     ↓
FastAPI (async)
     ↓
LangGraph Orchestrator
     ↓
┌─────────────────────────────────┐
│  News Agent    → latest news    │
│  Financial Agent → stock data   │
│  Risk Agent    → risk scoring   │
│  Report Agent  → write report   │
│  Critic Agent  → verify quality │
└─────────────────────────────────┘
     ↓
SQLite Database (audit logs)
     ↓
Structured Research Report
```

## Features

- **5 Specialized Agents** — each agent has one focused responsibility
- **LangGraph Orchestration** — conditional routing based on confidence scores
- **Real-time Financial Data** — yfinance + Tavily search combined
- **Critic Agent** — validates report quality before delivery
- **Confidence Scoring** — every report includes a reliability score
- **Full Audit Trail** — every agent action logged to SQLite
- **Async FastAPI** — non-blocking pipeline using ThreadPoolExecutor
- **Production Error Handling** — graceful failures with detailed error reporting

## Tech Stack

| Layer | Technology |
|---|---|
| Agent Framework | LangGraph |
| LLM | Groq (Llama 3.3 70B) |
| Financial Data | yfinance + Tavily Search |
| API | FastAPI + Uvicorn |
| Database | SQLite + SQLAlchemy |
| Re-ranking | Cohere |
| Containerization | Docker |

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| POST | `/research` | Research a company |
| GET | `/report/{session_id}` | Retrieve saved report |
| GET | `/health` | System health check |

## Quick Start
```bash
# Clone
git clone https://github.com/RaviKumarWeb/finsight-ai.git
cd finsight-ai

# Install
pip install -r requirements.txt

# Configure
cp .env.example .env
# Add your API keys to .env

# Run
uvicorn api.main:app --reload
```

## Environment Variables
```
GROQ_API_KEY=your_groq_key
TAVILY_API_KEY=your_tavily_key
COHERE_API_KEY=your_cohere_key
```
